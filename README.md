Web crawler, Used for crawling Ecommerce  websites in order to fetch data like product price, image, seller ....

## Installing / Getting started

A quick introduction of the minimal setup you need to get a hello world up &
running.

```shell
python -m venv venv
```

## Activate venv : 
Linux:
```shell
source venv/bin/activate
```

windows:
```shell
venv/bin/activate
```

## Installing scrapy 

```shell
pip install scrapy
```
